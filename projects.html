Web Controlled LED
Goals: The goal of this project was to control an LED light on an Arduino microcontroller through the web using Meteor.js; a framework to build web-apps.
Results: The Arduino was set up as a server and listened for HTTP requests coming in, processed the message, and turned an LED On or Off. From the browser, the interface contained an On and Off button and would trigger an HTTP GET request to the Arduino. The Arduino was connected through my WiFi network allowing me to move the Arduino throughout my house and even control the LED from my iPhone!
Technology: HTML, CSS, Javascript, Meteor.js, C++, Arduino Uno, Arduino WiFi Shield
